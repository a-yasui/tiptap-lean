# JS-TipTap

This template should help get you started developing with Vue 3 in Vite.

縦書きができる感じのエディターを TipTap で仮実装してみた。

site: https://a-yasui.github.io/tiptap-lean/

## 使い方

### インストール

```bash
npm install
```

### 開発サーバーの立ち上げ

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### デプロイ

```bash
npm run deploy
```
